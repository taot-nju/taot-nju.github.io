碎片知识分享

 问题  项目中间过程文档如何科学命名？

  • 推荐 ISO 8601日期 + 项目阶段 + 版本号 + 描述。例子：
     20260515_180619_exp1_data_preprocessing_v1.2_filtered_outliers.docx 。
    禁止使用“最新版”“最终版”“整理版”。每天结束时，删除或归档旧版本，只保留最新的。你可以把命名规则写在一张纸上贴电脑旁。

 问题  如何给变量命名？（看看Claude Code的命名！）

  •  # Real CC timeouts: 
  •  # TOOL_HOOK_EXECUTION_TIMEOUT_MS = 600000 (10 minutes for tool hooks) 
  •  # SESSION_END_HOOK_TIMEOUT_MS = 1500 (1.5 seconds for SessionEnd hooks)

 问题  Easy Connector 连上立刻 “已注销 VPN、会话已过期” 。 最简单优先试（最快见效）

  • 完全退出 Easy Connector，右键以管理员身份运行；
  • 清除缓存：软件里找「清除缓存 / 重置配置」，没有就卸载重装最新版；
  • 关掉电脑的代理、加速器、梯子、其他 VPN，只开校园 VPN。

 问题  一些常见软件的公共快捷键有哪些？（助力丝滑操作）

  •  Shift  + 鼠标滑轮 = 方便操作目标窗口的水平滚动条 (Horizontal Scroll Bar)
  • 按词删除（光标所在位置）
    •  Ctrl + Backspace ：删除光标左侧的一个完整单词。
    •  Ctrl + Delete ：删除光标右侧的一个完整单词。
    •  Alt + Backspace ：等同于 Ctrl + Backspace，主要在 Linux 等特定系统或部分终端中使用
